What is wrong with this SQL query?

Code 1:
SELECT Id, name, YEAR(BillingDate) AS Year
FROM Records
WHERE Year >= 2010

Code 2:
SELECT id, name
FROM students
WHERE grades =
             (SELECT MAX(grades)
              FROM students
              GROUP BY subject_id;);
